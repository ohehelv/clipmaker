# ComfyUI Workflow templates

В этой папке лежат JSON-шаблоны в формате **API** (Save (API Format) в ComfyUI),
которые загружает соответствующий генератор из `backend/generators/`.

## Как добавить/обновить шаблон

1. Открыть ComfyUI, собрать рабочий пайплайн для модели (например `animatediff`).
2. Включить `Settings → Enable Dev mode Options`.
3. Кнопка `Save (API Format)` → сохранить файл `<name>.json` сюда.
4. Открыть JSON и переименовать ключевые узлы (свойство `_meta.title` и/или
   `id`) так, чтобы они совпадали с теми, которые патчит генератор
   (`POSITIVE`, `NEGATIVE`, `LATENT`/`EMPTY_LATENT`/`SVD_LATENT`, `SAMPLER`).

   На уровне JSON узлы идентифицируются строковым ключом верхнего уровня
   (`"3": {...}`). Замени ключи на удобные имена (например `"POSITIVE"`,
   `"NEGATIVE"`) и не забудь обновить ссылки `[ "POSITIVE", 0 ]` в полях `inputs`
   других узлов.

5. Если поля у твоего workflow называются иначе (например `num_frames` против
   `length`), скорректируй `build_patches` в соответствующем модуле генератора.

## Текущие файлы

Все JSON в этой папке — **заглушки** с минимально валидным содержимым,
которое сразу упадёт при запуске. Это сделано намеренно: настоящие workflow
зависят от установленных custom-нод и моделей. Замени их своими экспортами.
