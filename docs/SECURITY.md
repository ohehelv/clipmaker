# Безопасность

## Минимальные требования

- Не публикуйте сервис в интернет без reverse proxy и TLS.
- Используйте `API_KEY` в `.env` для защиты API.
- Ограничьте `CORS_ORIGINS` только доверенными доменами.

## Секреты

- Не коммитьте `.env`.
- Храните `OPENROUTER_API_KEY` и `HF_TOKEN` только локально.

## Docker

- Запускайте с актуальным Docker Desktop.
- Регулярно пересобирайте образ после обновления зависимостей.
- Используйте только официальный образ Python и проверенные зависимости.

## Инсталлятор

- Используйте только HTTPS URL вашего GitHub репозитория.
- Для распространения вне доверенного круга подпишите `.exe` сертификатом.
- Инсталлятор работает без админ-прав и устанавливается в профиль пользователя.
